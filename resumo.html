Dicionário de Tags HTML

Abaixo estão as tags aprendidas durante as aulas e suas respectivas funções:

<html>

Tag principal que envolve todos os itens da página.

<head>

Local onde ficam as informações que não aparecem diretamente na página, como ajustes e programação.

<body>

Onde fica todo o conteúdo visível da página.

<h1>

Utilizada para marcar títulos principais no documento.

<p>

Utilizada para definir parágrafos de texto.

<br>

Insere uma quebra de linha; não possui tag de fechamento.

<img>

Utilizada para inserir imagens; também não possui tag de fechamento.

<ul>

Cria uma lista não ordenada (geralmente com marcadores de ponto).

<ol>

Cria uma lista ordenada (numerada).

<li>

Define um item dentro de uma lista ordenada ou não ordenada.

<dl>

Define uma lista de definição.

<dt>

Marca o termo a ser definido em uma lista de definição.

<dd>

Descreve a definição do termo em uma lista de definição.

<a>

Cria links para outras páginas ou partes da mesma página através do atributo href.

<style>

Utilizada dentro do <head> para adicionar estilos CSS à página.

<script>

Utilizada para adicionar códigos de programação JavaScript.
