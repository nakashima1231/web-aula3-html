Atributos Class e Draggable

1. O que é a Class? Diferente do ID, a class pode ser usada em múltiplos elementos. Ela é ideal para criar padrões de design que se repetem, como botões, cartões ou listas, permitindo que você altere todos de uma só vez no CSS.

2. O Atributo Draggable: Este parágrafo possui o atributo draggable="true". Isso informa ao navegador que o elemento pode ser movido pelo usuário. Tente clicar e arrastar este bloco!

3. Interatividade: Embora o HTML permita o movimento visual, para que o objeto seja "solto" em algum lugar e execute uma ação (como um sistema de arrastar arquivos), costumamos usar a API de Drag and Drop do JavaScript.

4. Experiência do Usuário: O uso de classes junto com elementos arrastáveis facilita a criação de interfaces modernas, como quadros Kanban ou organizadores de tarefas, mantendo o código limpo e organizado.
