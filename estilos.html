O Atributo Style (CSS Inline)

O atributo style é usado para aplicar regras de CSS diretamente a um único elemento HTML. Diferente das classes ou IDs que ficam em arquivos separados, o estilo aqui é definido dentro da própria tag, usando a sintaxe nome-da-propriedade: valor;.

Este parágrafo está sendo estilizado individualmente com o atributo style!

Uma característica crucial do style inline é a sua especificidade. Ele tem prioridade sobre quase qualquer outra regra definida em arquivos CSS externos ou na tag <style>. Isso o torna útil para correções rápidas ou estilos que precisam ser únicos e imutáveis.

No entanto, o uso excessivo desse atributo é considerado uma má prática em projetos grandes. Como os estilos ficam "espalhados" pelo HTML, fica muito difícil manter a consistência visual do site. Se você precisar mudar a cor de 50 botões, terá que editar 50 tags manualmente em vez de apenas uma linha no arquivo CSS.
